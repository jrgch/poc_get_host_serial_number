## Descripción

Este proyecto es una PoC muy rápida para evaluar la librería systeminformation, con el objetivo de obtener el serial del host con sistema operativo Ubuntu 24.04.2 desktop aunque el proyecto esté ejecutándose sobre Docker

## Para instalar las dependencias y ejecutar directamente en el host

```bash
$ npm install
```

## Compilar y ejecutar el proyecto

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Para desplegar en Docker

Antes desplegar el contenedor debe ejecutra la siguiente instrucción en la línea de comandos, esto permitirá compartir dicha ruta como un volumen en el docker-compose
```
sudo chmod -R 744 /sys/class/dmi/id
```

Si tiene virtaulizado el Ubuntu con VBox, puede establecer un serial con el siguiente comando
```
VBoxManage setextradata "jrgranada-VirtualBox" "VBoxInternal/Devices/pcbios/0/Config/DmiSystemSerial" "SERIAL-FAKE-12345"
```

Ejecutar 
```bash
$ docker compose up
```



## Consulta del serial del dipositivo

Para obtener el serial del dispositivo puede acceder al siguiente endpoint:

http://localhost:3000/system-information
